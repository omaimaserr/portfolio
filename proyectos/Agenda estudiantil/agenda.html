Módulos

Diseño de Interfaces Web

En esta asignatura se estudian las técnicas de diseño y desarrollo de interfaces web interactivas.

Profesor: María Carmen Armijo

Despliegue de Aplicaciones Web

Esta asignatura trata sobre el desarrollo de aplicaciones web dinámicas utilizando tecnologías modernas.

Profesor: María Leticia Riaza

Desarrollo Web en Entorno Servidor

En este módulo se centra en el desarrollo de la parte del servidor de aplicaciones web.

Profesor: José Alberto López

Desarrollo Web en Entorno Cliente

Aquí se estudia el desarrollo del cliente de una aplicación web, cómo interactuar con servidores y APIs.

Profesor: Miguel Ángel Cruzado

Empresa e Iniciativa Emprendedora

Este módulo cubre temas de emprendimiento y la creación de empresas en el sector tecnológico.

Profesor: María Pilar Alvarez

Inglés

En este módulo se estudia el ingles técnico en función al grado superior

Profesor: Elena Menchero
